# 🚀 60-Second Deployment

## The absolute fastest path to a live URL

### 1. Create the repo (30 seconds)

Go to https://github.com/new and create a new **public** repository named `pto-tracker` (or anything you like).

### 2. Upload the files (20 seconds)

On the empty repo page, click **"uploading an existing file"**. Drag in:
- `index.html`
- `README.md`
- `LICENSE`
- `.gitignore`

Click **Commit changes**.

### 3. Turn on GitHub Pages (10 seconds)

- Go to **Settings** (top nav of your repo)
- Click **Pages** in the left sidebar
- Under **Build and deployment**, set:
  - Source: `Deploy from a branch`
  - Branch: `main` → `/ (root)`
- Click **Save**

### 4. Wait ~1 minute

Your tracker will be live at:

```
https://<your-github-username>.github.io/pto-tracker/
```

Bookmark it. That's your permanent URL.

---

## 💡 Pro tips

- **Custom domain:** Add `pto.yourdomain.com` in Settings → Pages → Custom domain
- **Auto-backup:** Export the JSON monthly from the Export button (bottom right)
- **Multi-device:** Import the JSON on your other device to sync
- **Fork it:** Anyone can fork the repo, edit their hire date in `index.html`, and deploy their own copy in ~2 minutes
